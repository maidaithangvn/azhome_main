# AzHome Main Workspace

Workspace tập trung cho tất cả các dự án AzHome Xây Dựng sử dụng hệ thống AzHome SaaS.

## Cấu trúc thư mục
- `az_addons_cons/`: Chứa các module Odoo phát triển cho mảng Xây dựng (AzHome Construction).
  - `az_construction_management`: (Submodule) Module quản lý thi công.
- `az_addons_saas/`: Chứa các module quản lý SaaS (AzHome SaaS).
- `docker_build/`: Chứa các cấu hình Docker deployment.

## Cách làm việc
Mọi thay đổi và nâng cấp tính năng sẽ được thực hiện trực tiếp trong workspace này và đồng bộ lên GitHub.
